HT Politic is a Political WordPress Plugin. 

== Description ==
HT Politic - Political WordPress Plugin is a clean, modern and responsive design for Politicians, Ministers or Members of Parliament and event management. It is a great online solution for political parties, social movements, NGO’s, crowd-funding campaigns, and other politics related websites. It is a fully responsive and mobile friendly plugin to create unlimited campaign for the website. 
In addition, this plugin supports Elementor page builder. Using this Elementor Page Builder, you can show the campaigns/event anywhere in the site in a quick time. Creativity and effectiveness are all around in this WordPress Plugin.
Moreover, you can insert the options like Gallery, Slider and Timeline to embellish your theme with more functionality. Finally, showing the advisor and projects is very important for creating trust. You can add the advisor and completed projects in the site easily using this plugin. Last, but not the least, having blog post option is the core tool to generate traffic to the site. You can manage the blog posts with no hassle if you use HT Politic - Political WordPress Plugin.

== Installation ==
This section describes how to install the HT Politic - Political WordPress Plugin and get it working.

= 1) Install =

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "HT Politics ".
3. Install, then Activate it.

= Old Way: =
1. Unzip (if it is zipped) and Upload `wp-politic` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
1. After install and activate the plugin you will get a notice to install CMB2 Plugin ( If already install it then do not show notice. ).
2. To install the plugin click on the "Button" Install CMB2.
3. After install and activate the both plugin you will get a Dashboard Menu(WP Politic).
